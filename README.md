## 1. [Game]Stone Flower(석화)
엔진 : Unity 3D, 유니티   
플랫폼 : PC, PS5, Nintendo SWITCH   
> 개요 : Stone Flower는 스토리를 중시한 3D 액션 어드벤처 게임으로서 게임 이름인 Stone Flower을 직역한 ‘석화’를 컨셉트로 하여 
그와 걸맞게 전설상에서 석화를 사용하는 종족인 ‘메두사’와 돌을 조각하는 ‘조각가’가 주인공입니다.   
게임 내용 : 주인공 일행은 돌로 된 괴물들이 돌아다니는 이 세계를 헤쳐나가며, 세계 곳곳에 흩어져있는 4군데의 유적을 탐사하게 됩니다.
 세계 멸망을 막기 위해 움직이는 둘은 수 많은 퍼즐을 풀고 유적을 헤쳐나갑니다.
